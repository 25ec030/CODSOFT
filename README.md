# CODSOFT Projects

This repository contains small Python GUI utilities built as desktop applications.

## Included projects

- `01_todolist.pyw` — a simple task manager for adding, editing, and tracking to-do items.
- `02_calculator.pyw` — a basic calculator for arithmetic operations.
- `03_password_generator.pyw` — a utility for generating secure random passwords.

## Notes

Each script is designed for easy use on Windows and can be launched directly with Python.
